Grunt

An automation tool that aids development, testing, and optimisation.

Some of my favourite Grunt tasks:

  • Watch my code and inject changes into the browser on each target device at once grunt-browser-sync
  • Lint my JavaScript and enforce code standards as well as pre-empt errors grunt-contrib-jshint
  • Assess my CSS and insert browser specific prefixes, e.g. -webkit-user-select grunt-autoprefixer
  • Prefix asset filenames with a unique hash for anti-caching purposes grunt-rev
  • Automating minification and concatenation across the project grunt-contrib-uglify
  • Generating sprite sheets and CSS based on specified image files grunt-spritesmith
  • Compiling requirejs module driven applications into an optimised build grunt-requirejs
