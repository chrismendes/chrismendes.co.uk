Grunt

www.grunt.org

An automation tool that aids development, testing, and deployment.

In particular I enjoy using Grunt to:

  • Watch my code and inject any changes into the browser for quick development
  • Lint my JavaScript and warn of errors or inconsistency
  • Assess CSS and insert browser specific prefixes for styles
  • Prefix asset filenames with a unique hash for anti-caching purposes
  • Automating minification and concatenation across the project
  • Generating sprite sheets and CSS based on specified image files
