Grunt

An automation tool that aids development, testing, and optimisation.

Some of my favourite Grunt tasks are:

  • grunt-browser-sync - Watches code and injects changes into the browser on each target device at once
  • grunt-contrib-jshint - Lints JavaScript and enforces code standards as well as pre-empts errors
  • grunt-autoprefixer - Scans CSS and inserts missing browser specific prefixes, such as -webkit-user-select
  • grunt-rev - Prefixes asset filenames with a unique hash for anti-caching purposes
  • grunt-spritesmith - Generates sprite sheets and CSS based on specified image files
  • grunt-requirejs - Compiles requirejs module driven applications into an optimised build, optionally with almond.js
