The Basics

HTML5

First encountered HTML at an early age, worked with it throughout career. Strict on keeping the markup void of implied styling in favour of maximum semantic meaning.

Familiar with HTML5 - new elements, new input types, HTML5 video for example.

CSS3

Prefer to use the SMACSS model where possible for good modularity and scalability. Experienced at troubleshooting cross-browser presentation issues.

Proficient with media queries and CSS3 animation.

Sass / LESS

Equally capable with both, have used both commercially. Happy with nested styles and inheritance, writing variables, mixins, functions, etc.

JavaScript

Compentent object-oriented JavaScript developer - functions, objects, classes, prototype, etc.

Have experience with a wide range of projects with different levels of JavaScript involvement, from web pages requiring mere form validation to large complex webapps driven 90% by JavaScript.
